# Climate Risk & Disaster Management Dashboard

## Description
A web-based dashboard that displays climate-related risks such as flood risk and heat alerts to support disaster preparedness and response planning.

## Features
- Climate risk overview
- Flood risk alerts
- Heat wave alerts
- Disaster response guidelines

## Technologies Used
- HTML
- CSS
- JavaScript
